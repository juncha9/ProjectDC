Ruriweb.com Crawler
====================
이 프로그램은 루리웹 홈페이지를 크롤링하여 단어를 추출하고 빈도 높은 단어들을 WordCloud로 만들어주는 프로그램입니다.

Usage
====================

## 1. 파이썬

    1. python 3.6.5 (x64)를 설치

    2. 명령창에서 아래의 명령어를 실행

	python -m pip install --upgrade pip \n
	pip install pylint \n
	pip install requests \n
	pip install beautifulsoup4 \n
	pip install lxml
	pip install pandas
	pip install simplejson
	pip install pygame
	pip install git+https://github.com/e9t/PyTagCloud.git
	pip install konlpy
	pip install jpype1
	pip install openpyxl
	pip install xlrd
	pip install Pillow

     3. RULI.py 실행

## 2. 실행파일

    1. Ruriweb Crawler.exe 실행
